# Takasumi Stock News
- 株価情報を1時間に1度送信します
- 直前の株価、3日間平均との比較を表示します
- 変動幅が大きいものは強調して表示されます

## 使い方
1. Botをサーバーに導入します。[招待リンク](https://discord.com/oauth2/authorize?client_id=1490209013227323442&permissions=2048&integration_type=0&scope=bot)
1. 株価情報を表示したいチャンネルで`/subscribe`を実行します
1. 配信を止める場合同じチャンネルで再度`/subscribe`を実行します
